Project Name: UnityTest

Description:
This is a simple Unity game where the player collects points that randomly spawn on the map. The main objective is to gather as many points as possible within a given time or before all points are collected.

Main Mechanics:
- Points spawn at random locations on the map.
- The player moves around the map to collect these points.
- Each collected point increases the player's score.
- The game may include a timer or a win condition based on the number of points collected.

Design Decisions:
- Focus on simple, intuitive gameplay suitable for quick sessions.
- Randomized point spawning to encourage exploration and replayability.
- Minimalist design to keep the project lightweight and easy to expand.

Technologies:
- Unity Engine
- C# scripting

Future Extensions (optional):
- Add obstacles or enemies.
- Power-ups or bonuses.
- Multiple levels or maps.
